Dokümantasyon

Kullanım

Bir kullanıcı "ekle" iznine sahip ise ve etiketler resim sayfasında gösteriliyor ise, kullanıcı bu etiketleri "düzenle" simgesine tıklayarak değiştirebilir.

İzinler

İzinler kullanıcıların durumuna bağlıdur (misafir, genel, kullanıcı, yönetici, site yöneticisi). Yapılandırılmıştan daha yüksek statüye sahip bir kullanıcı(ekleme , silme) eylemlerini yapabilir.

Kullanıcılar formu göndermeden önce eklemiş oldukları ilgili etileti silebilirler.
