Documentación

Funcionamiento

Si un usuario tiene los derechos suficientes y si las etiquetas se ven en la página " picture " entonces verá aparecer un icono seleccionable para editar las etiquetas de la imagen.

Permisos

Los permisos se definen sobre los estatutos de usuarios (invitado, genérico, visitante, administrador, webmestre). Para que un usuario pueda efectuar la acción (añadido, supresión), debe tener un estatuto por lo menos igual al estatuto definido.

Por ejemplo, si se define que sólo los usuarios que tendrán el estatuto visitante podrán añadir etiquetas, entonces los usuarios que tienen los estatuos invitado y genérico no podrán subir el numero de etiquetas.

El hecho de restringir los derechos sobre los que puede suprimir etiquetas es efectivo sólo sobre la supresión de las etiquetas ya existentes. Si el usuario trata de añadir nuevas etiquetas, podrá suprimirlos antes de actualizar.
