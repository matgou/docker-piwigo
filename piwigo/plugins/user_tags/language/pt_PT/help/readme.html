Documentação

Uso

Se um Utilizador tem permissão de "adicionar" e se as etiquetas são exibidas na página da imagem, ele pode editar as etiquetas relacionadas clicando o ícone "editar".

Permissões Permissões

estão ligadas a status dos Utilizadores (visitante, Utilizador genérico, administrador, webmaster). Os Utilizadores podem executar os procedimentos (adicionar, excluir) se tiverem status de acordo com o configurado. Utilizadores

podem excluir etiquetas por eles adicionadas antes da ativaçãao da extensão, mesmo que você não queira que os Utilizadores apaguem etiquetas relacionadas.
