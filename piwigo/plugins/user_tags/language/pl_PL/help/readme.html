Jeżeli użytkownik posiada uprawnienia do dodawania i tagi są wyświetlane na stronie obrazu to może je edytować klikając ikonę "edytuj"

Uprawnienia

Uprawnienia są powiązane ze statusem użytkowników (gość, normalny użytkownik, administrator, webmaster). Użytkownicy mogą korzystać ze swoich uprawnień jeśli status ich konta jest wyższy niż ten skonfigurowany.

Użytkownicy mogą usuwać tagi, które dodali przed wysłaniem formularza, nawet jeśli nie chcesz, aby użytkowicy usuwali powiązane tagi.
