Документация

Использование

Если пользователь имеет разрешение на "добавление" и если теги отображаются на странице изображения то он может редактировать этот тег кликнув по значку "редактировать".

Разрешения

Разрешения зависят от уровней пользователя (гость, обычный, нормальный, администратор, веб мастер). Пользователи могут выполнять действия (добавить, удалить) если их уровень больше чем указан в настройках.

Пользователи могут удалять теги, которые они добавили до отправки формы, даже если вы этого не хотите.
