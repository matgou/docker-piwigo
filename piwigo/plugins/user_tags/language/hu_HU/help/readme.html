Használat

Amennyiben a felhasználó rendelkezik a szükséges engedéllyel, a kép oldalon a címkék sorban tudja szerkeszteni a címkéket az ikonra kattintva.

Engedélyek

Az engedélyeket felhasználói státuszokhoz kapcsolhatjuk (vendég, átlagos, felhasználó, adminisztrátor, webmester). Azon felhasználók tudják a műveletet végrehajtani (hozzáadás, törlés), akik a beállított, vagy annál magasabb engedéllyel rendelkeznek.

Az engedéllyel rendelkező felhasználók törölhetik a címkéket még akkor is, ha ez nem kívánatos.
