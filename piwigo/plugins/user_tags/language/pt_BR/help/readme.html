Documenyação

Uso

Se um usuário tem permissão de "adição" e se etiquetas (tags) são mostradas na página da foto, ele pode editar as etiquetas relacionadas clicando no ícone "editar".

Permissões

Permissões são associadas ao status dos usuários (convidado, genérico, usuário normal, administrador, webmaster). Usuários podem fazer uma ação (adicionar,apagar), se eles tiverem um status maior que aquele ao qual pertence.

Usuários podem apagar etiquetas que tenham adicionado antes via formulário mesmo que você não queira que usuários apaguem etiquetas relacionadas.
