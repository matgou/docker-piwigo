Lietošana

Ja lietotājam ir pievienošanas "add" atļauja un, ja uz attēla lapas parādās iezīmes (tagi), viņš var rediģēt attiecīgos tagus noklikšķinot uz "edit" ikonas.

Atļaujas

Atļaujas ir piesaistītas lietotāju statusam (viesis, vispārējs, normāls lietotājs, administrātors, webmāsters). Lietotājs var veikt darbības (pievienot, dzēst), ja viņiem ir augstāks statuss kā nokonfigurētais.

Lietotāji var dzēst iezīmes (tagus), ko tie pievienojuši pirms formas iesniegšanas, pat ja nevēlaties, lai lietotāji izdzēš attiecīgos saistītos tagus.
