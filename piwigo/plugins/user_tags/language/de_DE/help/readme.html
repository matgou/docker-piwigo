Dokumentation

Benutzung

Wenn ein Benutzer die Erlaubnis hat Tags hinzuzufügen und diese auf den Bilderseiten angezeigt werden, dann kann er die zugehörigen Tags, durch klicken auf dem "Bearbeiten"-Symbol, bearbeiten.

Berechtigungen

Berechtigungen sind in Verbindung mit dem Benutzer-Status (Gast, Allgemein, Besucher, Administrator und Webmaster). Benutzer können eine Handlung (hinzufügen, löschen) durchführen, wenn sie einen Status haben das Gleich oder Bessere ist als den eingestellten Status.

Die Benutzer dürfen ihre hinzugefügten Tags vor dem Updaten noch löschen, auch wenn Sie nicht möchten, dass die Benutzer diese löschen.
