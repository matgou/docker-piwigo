Документація

Використання

Якщо користувач має дозвіл "Додати" і якщо теґи відображаються на сторінці зображень, він може редагувати пов’язані теги, натиснувши кнопку "Редагувати" значок.

Дозволи

Дозволи прикріплені до статусу користувачів (гість, загальний, звичайний користувач, адміністратор). Користувачі можуть виконати дію (додавати, видаляти), якщо вони мають статус більше, ніж один налаштований.

Користувачі можуть видаляти теґи, які вони до дали до завершення заповнення форми, навіть якщо ви не хочете щоб користувачі видаляли пов’язані теґи.

Ви можете дозволити користувачеві додавати тільки існуючі теги, якщо ви встановите прапорець "тільки існуючі теґи".
