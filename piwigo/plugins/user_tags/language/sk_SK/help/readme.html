Použitie

Ak používateľ pridal práva a ak kľúčové slová sú zobrazené na stránke fotky, môže upravovať súvisiace kľúčové slová kliknutím na ikonu upraviť.

Prístupové práva

Prístupové práva sú pripojené k statusu používateľa (hosť, spoločný, normálny, administrátor, webmaster). Používatelia môžu pridávať, vymazávať ak majú status vyšší ako nakonfigurovaný.

Používatelia môžu vymazať kľúčové slová, ktoré pridali pred odoslaním formulára, dokonca ak nechcete používateľom vymazať relevantné kľúčové slová.
