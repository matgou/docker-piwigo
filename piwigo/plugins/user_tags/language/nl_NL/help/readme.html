Instructie

Als een gebruiker geautoriseerd is, kan deze labels toevoegen en verwijderen bij elke foto, klik daarvoor op de "edit" knop.

Autorisaties

Autorisaties worden uitgedeeld per soort gebruiker (gast, algemeen, gebruiker, beheerder, webbeheerder). Mits de autorisatie dat toestaat kunnen gebruikers labels toevoegen en verwijderen.

Pas op: gebruikers kunnen toegevoegde labels verwijderen, zelfs indien u dat niet wilt.
