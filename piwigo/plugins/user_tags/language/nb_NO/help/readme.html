Bruken

Om en bruker har "legg til" rettigheter og om tagger er vist på bilde siden, kan han redigere relaterte tagger ved å klikke på "rediger" ikonet.

Rettigheter

Rettigheter er knyttet til brukerens status (gjest, generisk, normal bruker, administrator, webmaster). Brukere kan gjøre tiltak (legg til, slett) om de har status høyere en de konfigurerte sine.

Brukere kan slette tagger de har lagt til før registrerings skjema, selv om du ikke ønsker at brukere kan slette relaterte tagger.
