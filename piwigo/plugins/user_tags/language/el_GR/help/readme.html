Τεκμηρίωση

Χρήση

Αν ένας χρήστης έχει άδεια να κάνει "add" και αν εμφανίζονται ετικέτες στη σελίδα της εικόνας, μπορεί να επεξεργαστεί τις σχετικές ετικέτες κάνοντας κλικ στο εικονίδιο "edit".

Δικαιώματα

Τα δικαιώματα συνδέονται με το στάτους των χρηστών (επισκεπτών, γενικά, απλός χρήστης, διαχειριστής, webmaster). Οι χρήστες μπορούν να κάνουν μια ενέργεια (προσθήκη, διαγραφή), εφόσον το στάτους των είναι μεγαλύτερο από την ρύθμιση που είναι για άλλους.

Οι χρήστες μπορούν να διαγράψουν ετικέτες που προστέθηκε πριν προχωρήστε στην αποστολή, ακόμα και αν δεν θέλετε τους χρήστες να διαγράφουν ετικέτες που σχετίζονται.
